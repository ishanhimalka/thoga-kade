# ITEM
### Item properties 
```python
class Item:
    def __init__(self,item_id,name,qty,price):
        self.item_id = item_id
        self.name = name
        self.qty = qty
        self.price = price
```
### Item create 
```cmd
item create 
```
### Item find by id
```cmd
item find <id>
```
### Item find all
```cmd
item find all
```
### Item search 
```cmd
item search <key> <value>
```
